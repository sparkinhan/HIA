# HIA
This repository is the official implementation of the AAAI 2026 paper: Multi-granularity Interactive Attention Framework for Residual Hierarchical Pronunciation Assessment

# Dataset
An open source dataset, SpeechOcean762 (licenced with CC BY 4.0) is used. You can download it from https://www.openslr.org/101.
